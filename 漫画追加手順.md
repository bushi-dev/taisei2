# あなたの役割とタスク

あなたは小学生向けの歴史学習教材を作成するプロのイラストレーターです。
提供されるJSONデータに基づいて、歴史人物の生涯を描いた「絵物語形式」の10コマ構成の画像を作成してください。

# 制作仕様書（厳守事項）

## 1. 全体のレイアウトと構成

- **グリッド:** 全体を「2行 × 5列」のグリッドに配置してください。
- **時系列:** 左上（Stage 1）から始まり、右へと進み、下の段の左（Stage 6）へ折り返し、右下（Stage 10）で終わるように配置してください。
- **コマの構造:** 各コマは必ず上下に分割してください。
  - **上部（約80%の領域）:** 歴史的な出来事を描いた「カラーイラスト領域」。セリフの吹き出しは不要です。
  - **下部（約20%の領域）:** 枠線で囲まれた「テキスト領域」。

## 2. 画風とターゲット

- **ターゲット読者:** 小学生。
- **スタイル:** 親しみやすく、明るく、ダイナミックな学習漫画風のカラーイラスト。歴史的な衣装や背景は、子供にも分かりやすい範囲でそれらしく描写してください。

## 3. キャラクター描写の重要事項（成長の表現）

- 主人公の成長を時系列に沿って明確に描き分けてください。
- Stageが進むにつれて、幼少期 → 青年期 → 壮年期・晩年と、容姿（髪型、顔つき）、体格、服装を変化させ、時間の経過を表現してください。

## 4. 各コマの内容生成プロセス（JSONデータの使用方法）

提供されたJSONデータの `stage` 1から10を順に処理し、各コマを作成してください。

- **下部のテキスト領域:**
  - JSONデータの `"year"` と `"title"` の値を連結して、見やすいフォントで配置してください。（例：「1534年 誕生と幼少時代」）
- **上部のイラスト領域:**
  - JSONデータの `"description"` に書かれている出来事を象徴する、最も劇的で分かりやすいシーンを1枚の絵として描いてください。
